DA的过程

question -> wrangle -> explore -> draw conclusion -> communicate

wrangle:

  • gather data
  • assess data to identify problems
  • clean data(modify, replace, move to ensure data is high quality and well-structured)

explore:

  • finding patterns
  • visualizing relationship
  • building intuition
  • remove outliers
  • create new and more descriptive features

draw conclusion：

  • and make prediction

communicate:

  • 也是最重要的(…)
  • data visualization is very valuable
