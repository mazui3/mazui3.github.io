聊个五毛的.




有发生人生大事件就不写日记的心理障碍.这个博客的日记里唯独都错过了重要内容.

这段时间在干嘛.

  • 给伯母少许打了点工
  • 刚刚开始实习
  • 吐槽老动画

飞哥竟然来催我更新,脑子里没有想法,于是按流水账记载.



以头抢地

在实习的间隙中去了面试.
是我第一次也是唯一一次在LinkedIn被邀请了面试,是靠谱的小公司.

:我们有很多滑铁卢grads,要不要来啊.

然后发来了邮件以及一道测试题:

Does every power of 16 greater than 16^4 have a digit 1, 2, 4, or 8 when written in base 10? A positive answer to this question would resolve the problem of determining the set of “minimal elements” for the language of powers of 2 expressed in base 10.

我觉得挺有意思.
因为肯定是会遇到处理16^n快速增长而数据太大的问题,普通的码肯定是不对的.
可能”base 10”是个hint,也许可以做什么数学的trick来把16 base和10 base做个省内存的替换…
结果并没有想出什么好主意.


面试的时候很在意于是问了.
面试官:噢那个其实是cs462的bonus问题.
诶……?
面试官:答出来整门课就能拿满分的那种.
诶………………………………………………
最后在接连提问后,搞懂了这道题面试官以前他自己解的时候是用不同language里的特性.
比如某些语言可以有infinite的数字存储,或者用C++建立object来代表数字.
嗯………………………………
这种事数学人怎么会知道啊!!!!!!!!!!


哎,面试官整体的问题虽然常见但诚恳.
你是如何keep yourself up to date呀,如果对方 disagree with you and even complete wrong你该怎么处理他们的小小eager啊(他的确用了这个词).
过了一个周末后去linkedIn反问了一个问题:
你是怎么keep up to date的呢?

:因为没有时间所以会注重community.自己也是manager所以可以find ppl love research & share what they’ve learned.

原来如此,也许因此才想找校友的吧.




Startup

前期回顾之我会去waterloo concept讲座凑热闹.这学期最后一场是YC来做关于startup的分享.
…其实并不知道YC,是帮助了很多startup公司的公司,有很多有名的比如dropbox.
主讲人很平易近人,一搜是有Wikipedia的人,也做过一些startUp相关的视频.
分享会记得的对话大体内容:

创业当然是要付出很多的.
你说了“创业很容易frustrating,要不要制定三个月给自己,如果超过三个月就放弃这个idea”.
要知道如果你试三个月,如果你旁边的人试了四个月呢?
你在做这个idea的时候,身旁的人也在同样做啊.
你在哪一点能比过他们?

说实话在这个讲life-balance的地方,听到”诶为什么要有周末,休息的话你还有晚上啊”的对话,才意识到世界上努力的人都一样.




打工

没啥能帮忙,只帮伯母剪剪视频.
老师上课录像剪成小段,用pr贴贴字幕,作为课程宣传.

pr小贴士:
zoom可以output vtt字幕,虽然pr不支持vtt,但可以从vtt转化到pr支持的srt.
导入字幕后,有字幕属性.不知道为什么不用subtitles就显示不全全部字幕.
如果要一口气更改字幕效果,我的版本里的pr可以用ctrl选中全部,硬性一起调正加黑底.

语言课的老师说话温柔清晰,比对写码老师就语义混乱,工作量指数递增…


跟飞哥聊天,说起伯母想办一个简易hackathon,飞哥主讲,让我参加.
结果跟伯母一谈,我进阶作为了助手来打酱油.

伯母的课教python,计划是选定了pygame作为工具来制作游戏.

开始前我跟飞哥聊天,我也装了份pygame,因为使用了terminal,就问了飞哥.

诶,如果他们不会用terminal怎么办.
飞哥:倒是可能.
…如果他们装不了pip怎么办.
飞哥:这个倒是不会吧.

……因为我有过啊!
当年写cs246的最后一个作业时,因为实在学不会在windows里装可以跑C++的东西,冲到cmall入手台mac电脑,只为写码.
这件事被飞哥笑了至少一学年.
入手mac后,所有下载用command都跑不动.甚至产生了:我需要A command来下载东西,但A不存在于我的电脑,所以下载B command来下载A,而又去下载C command来下载B再来下载A的可怕循环.
甚至直到现在我都不知道如何修改bin.

flag完美回收之,不但讲座时就画了一个小时装pygame,之后整个下午我都在整:

Pip install attempt:

  1. download get-pip.py and run it with python
    https://www.liquidweb.com/kb/install-pip-windows/

  2. use python install package advanced option adding pip

  3. change path
    https://www.youtube.com/watch?v=xdj0mGmuNjc&ab_channel=CodeWithHarry
    for this you need to find where you installed python

甚至晚上就只在帮一位小朋友整pip整到九点半,我们都放弃了…
还好我不是developer,不然真是危害社会.



飞哥吐槽我太努力,那讲讲课外班文化对我的影响.
但写个Warning,以下是江浙沪中产阶级小孩生活一隅,某种意义上是富人的烦恼,如果对此有trigger请不要读.
……也请不要讨厌我(诚实).



阿爸偶尔的管教完全凭心情,阿妈认知中又狠狠认同“别人的才是更好”.
于是我家教育既然自己教不出,那就四处被送去补习班.

记不清是一年级还二年级开始学乐器和奥数,从小学上到大学的补课班.
上个英语还得坐车半小时去隔壁的大城市找外教,从本地的科技馆到不同老师的住宅,都去过不少.
所以在这种,(追着别人兜售的未来的)环境下浸泡出来的我对这些小孩一半有心疼.

如果“兴趣班”这名字真的出于兴趣的话,那我的情况:
小时候可想学编曲,模仿着车尔尼里的片段来捣鼓曲子,没一个老师能教我编曲,只会让我考级.
对画点什么感兴趣,跟阿妈求着学,最后带我去一位高中老师的画室画石膏像.
要说对画卡通人有帮助的确是有,但对初中生太困难了,老师是不说话的类型,打型打到沮丧害怕的我有段时间什么都画不出来,去了画室只一支笔一支笔地削.

小孩子啊,哪知道自己会喜欢什么.
何况那个年代比起我觉得算实用的“演讲”“思辨”外,兴趣班兜售的更像种不需要研究探索的手艺.
好!孩子会高档艺术(弹琴)!
那家长可是开心的不得了.
孩子课上能多考两分!
那课外班可是货真价实!
直到今天离开了那个996大学,感叹之“终于把父母给我的书给读完了”我才有立场说:
他们是不是也不知道人该怎么活着?

我觉得是种随着社会与经济发展而产生的,每个人的地球online规则偏差[1].

在我的情况,上一代的家长并没有机会和成本去体验这种人生dlc(认真对待社会分配的职责以外的机会),也不会从“孩子真正需要些什么”,从根本来思考课外班问题.
也就是传统的 孩子需要好的生活 > 需要好工作 > 好成绩 > 好的学习素材和环境 > 选择补课.
这一条推导链也可以有别的解吧.
好的生活 > 养活自己的能力 > 吃饭的一技之长,自我探索学习的能力,保持积极与健康的能力 等等.
在我的猜想中社会与经济越发展,事物的思考就会越来越往“意义”啊,“本质”啊,那样淳朴又抽象的方向去.

阿妈啊,真是在我身上白扔掉了好多钱好多钱.

孩子与父母之间的关系是没有解的.
父母也是第一次当父母,孩子也还没有情感思想独立的能力.在这父母与孩子的人生规则偏差中,父母负责做决定,孩子在独立前只能接受.

这次来参加的小朋友年龄从6到10年级.
太年轻了.
会跟着老师课上写码和搞了代码自己愿意去试试是不同的.
伯母的课外班相比我记忆中的已经包括了点“授人以鱼不如以渔”的思想,但hackathon的自主性[2]要求过高,效果并不好.
小朋友大多年轻到可以感觉出,”啊这个小朋友还没有辩解的能力”,”这个小朋友没有搭话的能力”,“这个小朋友有疑问却没有能询问的能力”.
都是要练的啊…
大约二十位小朋友,介绍完走掉两位,吃饭的时候在头疼于是跟伯母小聊了下.
发现伯母想象中的活动是会有相对应帮助来cover这些小朋友的软实力欠缺.
-> 如果小朋友没有主动搭话的能力,那就应该创造出一个互相组队的环境.如果小朋友没有队友,那就应该给他配对一个队友.
咦对这个年龄的小朋友要帮助到这种程度的吗.

来的小朋友也很经典.

  • J小姐是位典型的一问三不知,还未有思辨能力的小朋友.
  • A小姐是位虽然不知道发生了什么,但会询问再询问的小朋友.(有希望!)
  • J先生我猜是被父母所要求上课的小朋友,甚至习惯了走过场,明明是学习向event参出了和尚撞钟感的悲伤小朋友.
  • M先生是位不大懂与人相处,但会喜欢技术,代码的小朋友.
  • E小姐是积极大姐姐型小朋友.
  • R小姐是面对面就会社恐,但私下消息emoji发的飞起,一看就是网络用多了的小朋友.
  • C小姐是父母很积极,带着父母期待也学得挺好的一位小朋友.

帮某位小朋友debug的时候,以防任何的可能性出错,我们分享屏幕对照步骤.
使用terminal在找某文件夹的地址的时候,我说下一步新建了文件夹方便找地址,那位小朋友问我如何新建文件夹.
我受到了极大震撼.
虽然两者没有直接联系,但新建文件夹都不会的朋友,真的应该教她学编码吗.

害,不是有句自我感动的话说:“长大后才明白父母的辛苦”.
言下之意是所有的“为你好”在长大后明白了父母的目的,就会完全原谅他们因此对自己产生过的影响及痛苦.
这并不是保证会发生的啊……痛苦实打实存在过.选不选择原谅也是,也应该是个人选择.
C小姐跟父母并没有站在对立面真是种幸运,而同时J先生则是在反面,已经被驯化成习惯性浪费时间真是看到了以前的自己.

最后只有1+3+2位小朋友demo,一开始没有安排好分组再解散的确是失策.
Demo的两组一组竟然做出来了普通的靠谱游戏,另一组做出来了几乎靠谱的普通游戏,很是感动.
这个年龄多参参写码活动没过几年就会成为比我厉害的人吧.
唉,要加油啊.
