背景素材收集

深海の北極星 - 素材おすすめサイト紹介

・きまぐれアフター
言わずと知れた超有名サイト。
商業レベルのハイクオリティな背景が多数。
ノベルゲーム作るならここはまず欠かせないでしょう。
有名すぎて他のゲームと被りまくる事だけが難点。

・消失点
ここも商業レベルのクオリティ。
現在はもう更新してない所だけが難点。

・KNT graphics
またまた商業レベルのクオリティ。
学校系の素材、特に学食は他にあまりないので便利です。

・空彩～コレクション～
空の素材ならココ！！
朝昼夕晩、いろ～んな空があって、しかもクオリティ高いです。
視覚的にバーンと魅せたい人におすすめ。

・かじじ帝國
田舎、自然系の背景が多いです。クオリティも素晴らしい。
古いのは夕方とか夜の差分がないですが、非営利ならば加工OKなので自分で作りましょう。
「さよなら、リアル」制作にあたって非常にお世話になりました。
田舎とか海とか森とか出したい人におすすめです。

・あやえも研究所
写真をイラスト風に加工した背景が多数置いてあります。
他のサイトでなかなか見つからない背景を補完するのに便利です。
「五月雨の記憶」「冬時雨の記憶」ではここの背景をメインに使わせていただきました。

・Moon☆Wind
クオリティがYES！高須クリニック！
最近現代背景が追加されたみたいです。
注目は校門の春夏秋冬差分！
こういう差分は非常に珍しいですね。
また、BGMやSE素材も揃っています。

・ヴィントドルフ
老舗サイトなのでちょっとサイズが小さいですが、waifu2x等で拡大して使えば無問題。
洋風とか和風とか、ファンタジー風の素材が揃ってます。
残念なことに、現在は更新されていません。

・K’s Factory
色んなシーンの背景が揃っています。
ゲームセンター、ファミレス等は他にはあまりないので便利。
ここも現在は更新されていないようです。

・mao space
サイズがちょっと小さいですが、他にあまりない素材が揃っています。
これが欲しいのに無い！　という時に覗いてみてはいかがでしょうか。

・Faraway
数は少ないですがクオリティが高いです。
映画館、水族館の背景が欲しい方におすすめ。

・しじみのおすまし
ここも数は少ないですがクオリティ高いです。
ファンタジー系のゲームを作りたい方におすすめ。

・異教徒たちの踊り
ティラノビルダー公式のサンプルゲームに使われている素材です。
数少なめですがクオリティ高し。
男子トイレの素材が欲しい方（！？）におすすめ。

・Peter
例によって数は少ないですがクオリティ高いです。
どこか幻想的な雰囲気を感じさせる現代背景です。
光の表現が上手いと思います。

Map Navigation

Map Navigation in RenPy | Vlog #4 Creating a Map Screen



行内字体描边

转载自Ren’Py 无上限！

在同一行内设置不同的字体描边。
因为文本样式标签不继承 outlines 属性，所以需要特殊的方法实现。

原理是把需要改变描边放到 Text 中，这样就能设置单独的 style。为了实现慢速显示，这里为每一个字都包装了单独的 Text，并且外包 Fixed 来控制字间距。



等待一定时间后触发特殊剧情

某些隐藏剧情的进入条件是在某一处对话处等待一定时间，可以通过这样的方式实现。

原理是设置一个超时自动跳过的对话，并记录对话前后的时间，对比时间差。
使用renpy.get_game_runtime，不会计算在菜单界面或存档期间的时间。



存档钩子

有的时候我们需要在玩家存档、读档时进行一些额外的操作，这就需要拦截FileAction这一行为。

简单解释一下上面的代码。
首先我们定义了一个ActionHook类，它的作用是包装一个Action并改变它的行为。可以看到它在创建时需要传入一个action和一个func，当获取属性时，返回action的相关属性，但调用（__call__）时改为调用func。
然后我们替换掉了原本的FileAction行为，改为创建一个ActionHook，其中调用由do_action执行。在do_action中， 进行是否执行原本行为的判断。
理论上用这种方法，可以拦截绝大多数的内置 action。



（数据删除）效果

利用负字偶距将大量文字压缩到同一位置，达到（数据删除）的效果。
同时，设置 noalt 和 alt，保证历史记录中可以正常显示原本的文字。



在立绘上用气泡标识当前说话人

注册全局对话钩子，每当新的对话发出时，获取当前说话人立绘的位置，叠加一个气泡图层上去。
此方法需要在创建 Character 时，指定其 image 属性。

此方法需要有一张和角色立绘大小相同的 speaking.png 作为气泡图像。
