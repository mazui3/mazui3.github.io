Children's Book Illustration - 作业

Posted on 2023-03-08 Edited on 2024-03-31 In 绘图笔记 Disqus: 5.5k 5 mins.

(什么竟然还还有的奇奇怪怪作业)
(博主也很惊讶!!)

此处混合了色彩和绘本两节课的份.




自画像

色彩课的第一个内容作业画自画像.要求先选配色,再画用喜欢的和不喜欢的颜色上色.
自画像好纠结啊…
不喜欢画自己的脸也不擅长画,硬画还会不可避免的二次元.
于是选择意译一下!

有一点 分散开来暴露给人看 的感觉,但也(可能)会不舒服吧.个人还是挺喜欢的(恶趣味).



Story Chess

色彩课的结课作业竟然是自主选题.
故事和棋局联系在一起是挺久之前的主意,写了下理出来思路,(感谢朋友的建议!),没有磨出一个合格的机制…只草草做了地图交作业.
说实话游戏机制真的得重头想过…但可以唠一下家庭组的房间.

  • 地下室杂物 + 老兔
  • 一层客厅 + 厕所 + 洗衣 + 书房 + 丹
  • 二层阳台 + 厕所 + 宾果

老兔住地下室是很早之前就决定的(08年?).老兔喜欢夏天开爆空调蜗在房间里打美少女游戏(Galgame).
最初版本的房间分配不一定找得到…
区别在于宾果一开始不是【人类】状态,而是作为宠物的【植物】盆栽,所以屋子没有二楼更没有宾果的房间.
保留下来的痕迹是宾果人设的头绳是绿叶,那个叶子才是本体.
丹和客厅都在一楼没有变.
丹是人类,学生需要上课,兔不是人类所以当无业游民.
现在想想世界观好怪,家庭收入是怎么来的.紧急打补丁丹是博士生上课做研究有工资.
不要在小学生(双重意义)写出来世界观里寻找真实感啊.

老兔的房间 - 床是吊床,不知道为什么就是吊床.地毯是圆形的.
电子设备都是老货,杂乱怪东西贼多,布局像中东集市店铺.
丹的房间 - 普通极简单色,床是普通四脚床.木家具,喜欢点香薰蜡烛.
宾果的房间 - 彩色喜欢贴装饰,床是两层抽屉床.有书柜一体书桌,书籍图集绘本居多.
虽然没有老兔那么夸张,也会收集一些曲奇盒,亮彩纸,时尚杂志之类的.

顺便一提旧版的老兔的地下室是自己挖出来的,挖的方向不对,挖到隔壁邻居家地下,而邻居是隐姓藏名的猫偶像.



角色

绘本课第一次作业就要求角色五视图 + 表情.
感觉在考动画系.
为了不画二次元(握拳),拿了几年前没做的game jam设计 - 鼻涕泡先生.
这个角色是否有在博客记载……并不不记得,于是前情提要:

手机点击 + 体感(?)游戏. Flappy bird + 平衡小球.
第一种stage是点击让角色靠鼻涕泡上升来躲避城市障碍物.
第二种stage是手机平放来把角色当盘子里的球,来穿过窄小的通道.



回到作业的表情.

五视图在舒适区,但表情在能力外了…!!!
可恶啊我的弱点…
没思考人物性格就摸表情挺不make sense的…所以只乱探索.
阿约什么时候来画表情神态六十天.



背景

作业课题原文是“environment”,更像是整体氛围的意思.
儿童绘本里的画风是挺多样的.而且很多很擅长用不同的texture,画材的或者画风的.
俺没有,俺真的完全没研究出texture,以平日插画的感觉摸的图.
术业有专攻…
而且觉得儿童绘本的绘图,跟我平日里的插画在【事物的focus point】上有很大不同.
绘本会添加很多对于插画来说无意义的元素,有点堆积得过多.
绘本的构图可以形成guide line,而插画(对于我来说)不是事物形成guide line,而是【服务于focus point】.
像绘本那样画插画的话,对我来说…一开始挺难受的!



文字

主题是lie.

这三张…是san0的成果(?).
上课前一天是色彩课的ddl,12点正正好踩了线.没有像大学时代那样踩完一个零点再通宵一个赶晨线.
成年人直接睡觉了…
于是变成了五点从单位下班,六点半就要交作业的极限一个半小时.
俺捏,在高中的速写夜校学了一招叫blind coutour.
不看画纸,直接盯着写生体来盲画.
因为肯定会型飞起不需要准确度,还有点形状,画得还快,是娱乐性很高的技能.
于是打算是在路过人多的地下通道盲两张“生活感”的+文青体自我感情叙述.

一开始坐在金融街地下的沙发写路过路人的鞋.写了两张.
人走的太快不是很好写,于是一边写一边往地铁走.

这一张的背影是…跟在别人背后一边走一边画.
因为平行速度还好,所以挺稳.

然后上了地铁,地铁不是很好写.
地铁写了两张.一张站台一张地铁内一张,我写空间的能力很烂.

最后在学校旁边的小咖啡店写一张,树画了两次.

然后逃离搭话的路人逃到另一家小餐厅,把纸头铺在桌上开始想词.

lie是这样的:明明主人公在郁郁输出,但画面却往上积极了一点.
词试着写了两次就定下来了.然后极速上色草完成度.
用的蓝笔和黑笔还是第二节课要用画具而去百元店拿的便宜货…

说实话这份作业比起成果,过程反而印象更深.
开始速写就会进入【世界也不过是线和弧度的组合】的中二模式,羞耻心也好,麻烦到周围也好都给我等一等,这个型写不下来了.更别说还有时间的限制.

大概是把自己都吓了一跳的程度吧…!



分镜

结课作业是32页的绘本分镜.
听到32页的时候被这个数字吓到.
为了弥补小时候没有看过真正儿童绘本书的缺失,跑到社区儿童绘本图书室爆看十本.
留意了一下页数,大部分真的是正好32页.
这个数字应该是业界的标准.

对我来说最难的是台词.

爆看的几本里 - 虽然是随便选的 - 儿童文学太阳间了.
太阳间了!!!
也有故事有悲伤的情节,整体感情都是向上的.
从去年(前)提出角色是否有寿命开始…就在思考到底故事是什么啊?
在故事中说谎 - 与我自己对故事的相信…产生矛盾!
我(阴暗b)是否能在不说谎的情况下 - 写出阳间的故事吗!

宾果小姐的职业idea:

  • 旅行家
  • 收银员
  • 歌手
  • 糕点师
  • 飞行员
  • 魔法师…

然后丹是化学家,老兔是艺术家(噗嗤).

有参考“一百位孩子的未来打算”之类的视频.
感叹之纯真跟大人世界混杂…的答案.故意没把医生,律师,老师写进去.
(丹和老兔的职业设定是当场敲的,丹转正了,老兔反正一如既往不务正业,T恤收集艺术家也算艺术家吧.)

我能写的阳间程度这可能就是极限了.

词打了一晚上,分镜两小时…分镜小天才!(得意忘形了对不起).
插画功底还是在的!就是我以外的人八成看不懂(靠).

最近没有时间真正的把它画完,四天赶出来的东西…也并不是很相信.



因为这次画的很快(羞愧的粗糙)没打算把分镜拿出来说,但跟朋友聊天说起,就聊一下.
阿约构图会注意把思考先放在【感情/形容词】上.

举例:
内容,宾果小姐要当收银员.
思考,宾果小姐要(因为数很多东西会快乐)(要快乐得意的)当收银员.
再举例:
内容,阿妹的背影.
思考,阿妹(在阿哥的记忆里)(包括了抓不住的过去,的感情)(也许在两人约定的地点)的背影.

而这些因为一开始纳入了考虑,所以会影响人物的比例,场景的安排,颜色的选择…也就是构图会比较跳比较脱线.



第一张因为是标题,所以是白底小人.
从第二张开始后脑内都有背景/颜色.
有刻意努力

  1. 每张插画的重心放在不同的地方.
  2. 前后的插画人物大小不同.
    是从爆读的绘本中总结来的,也许是绘本是给儿童的,动感会大一些,所以几乎每张都有这样的区别.

1+2 ->
书籍信息+标题.宾果小姐穿着大人衣服.
3+4 ->
宾果小姐思考未来.
上一页角色在左侧,所以先把角色画在右,再画出教室.
发现背景单单教室太闷了,于是把墙壁打破画了树林(或者放画框之类的).
5+6 ->
宾果小姐想环游世界收集小饰品.
把人物画小,再加上不同形状的大陆来说明环游世界.
饰品还加在大陆底下还是人物之上纠结了一下.
7+8 ->
宾果小姐喜欢清点物件所以想当收银员.
为了展示喜欢很多很多不同的物件所以构图很挤,直接让物品飘起来.
物品有大小区分,一来是有飘起来的感觉,二来是同一个size会无聊…也会太挤掉角色.
9+10 ->
宾果小姐想成为自作曲的歌手.
上一张动,这一张想安静一点,脑内是一束光打破黑暗,直照到宾果身上.因为台词中还有自作曲,于是加了点装饰性音符.
11+12 ->
宾果小姐想当糕点师.
安静过了是我的礼节,俺真的喜欢都给我动起来.
所以直接把宾果的烤盘给打翻了.
……虽然对于糕点师来说不大make sense!
13+14 ->
宾果小姐想当飞行员.
这张画了两次也不喜欢.
15+16 ->
宾果小姐想当小女巫.
简单的从城市上方飞过的构图.
因为下一张是很嘈闹的构图,所以在右侧(假设是右翻)加了大号星星作为过度.

17+18 ->
宾果小姐很纠结.
构图很简单是聚集了之前的六个职业.
中间的纠结宾果小姐不能画对称因为会卡在书籍翻页的中间,于是稍微往左斜一点.
19+20 ->
哥哥来打圆场.
叙事好难画…就拆成三个小构图.
绘本中有看到拆成小构图的构图也不少.我个人不喜欢在插画中加像漫画一样的拆分…那可以干脆画漫画,所以用了圆格子.
21+22 ->
在做化学实验的丹.
为了表现丹的确很认真,脑子里是想前景布置不同的玻璃烧杯试管,反射出很多丹认真的表情,来体现困恼和集中.
23+24 ->
把T恤铺一地的兔.
兔那么开心懒散,那就快乐躺平好了.
为表现东西一地,所以是俯视.
25+26 ->
丹说关于职业自己尝试了很多,但只有这个适合自己,于是就坚持下去了.
我不擅长抽象题材…
想的是既然尝试了很多,那就画很多个丹(其他职业),只有一个往前跑(化学).
27+28 ->
兔没有考虑太多未来,他是想到什么就去做的活在今日类型.
因为他得意洋洋所以有老兔的大头占半个屏.
右半边…脑子里想的是老版电脑大富翁有金银财宝从天上掉下来,然后玩家左右操作去接…老兔左右去接兴趣,的构图(梗没人懂).
29+30 ->
结语是you are the million(你有无数种可能性), you are the one(你也是当下独特的你自己).
一种祝福……但没有解决任何问题的结尾!!(
31+32 ->
老兔还在得瑟,丹已经在背后整理宾果刚穿的六套衣服了…



跟老师解释的时候,把宾果和哥哥们的玩偶拿出来讲设定.
老师给建议的时候很自然的说可以带着哥哥玩偶一起加入职业选择,以及大人宾果来回顾幼时的思考之类的.
第一次他人如此自然的接受了自家的设定,并且给出了建设性建议…这样的体验神奇极了!很感激.

有机会的话…再纳入给老兔的情书备选案.
